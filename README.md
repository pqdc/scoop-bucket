# pqdc Scoop Bucket

Scoop bucket for the pqdc governance toolkit — tools that make AI
collaboration auditable and trustworthy.

## Setup

```
scoop bucket add pqdc https://github.com/pqdc/scoop-bucket
```

## Available Tools

| Tool | Install | Description |
|------|---------|-------------|
| pqcc | `scoop install pqdc/pqcc` | Governance dashboard for pqdx state files |

More tools shipping soon: pqsn, pqit, pqgd, pqsl, pqvl, pqsq, pqfg.

## Quick Install

```
scoop bucket add pqdc https://github.com/pqdc/scoop-bucket
scoop install pqdc/pqcc
```
